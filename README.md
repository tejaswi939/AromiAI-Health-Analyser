# 🧠 AroMi AI – Health Analysis Assistant

AroMi AI is an AI-powered health assistant that analyzes user queries and provides real-time health insights using OpenAI API. The application features an interactive chat interface and a Flask-based backend for seamless communication.

---

## 🚀 Features

* 🤖 AI-based health query analysis
* ⚡ Real-time response generation (<2 seconds)
* 💬 Interactive chat interface
* 🔄 Session-based conversation handling
* 📊 Intelligent and contextual responses

---

## 🛠️ Tech Stack

* **Backend:** Python, Flask
* **AI Integration:** OpenAI API
* **Frontend:** HTML, CSS, JavaScript
* **Tools:** VS Code, Git

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/yourusername/aromi-ai.git
```

2. Navigate to project folder:

```bash
cd aromi-ai
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Add your OpenAI API key in `.env` file:

```env
OPENAI_API_KEY=your_api_key_here
```

5. Run the application:

```bash
python app.py
```
